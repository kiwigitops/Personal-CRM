# Dashboard

The dashboard is the main daily view.

It includes:

- Total contacts
- Warm contacts
- Overdue follow-up count
- Interactions this month
- Today and overdue follow-ups
- Suggested actions
- Stale contacts
- Recent interactions

## How To Use It

Start with overdue follow-ups. Then check stale contacts and add a lightweight interaction or follow-up.

## Limitations

Suggested actions are deterministic and based on simple CRM data. They are not a full AI assistant and should be treated as prompts, not truth.
