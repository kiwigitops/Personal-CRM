# Getting Started

Personal CRM helps you keep track of people, notes, follow-ups, and relationship context.

## Sign In

Local demo account:

```text
owner@personal-crm.local
password123
```

Use demo credentials only in local development.

## First Workflow

1. Open the dashboard.
2. Add or import contacts.
3. Open a contact.
4. Log an interaction.
5. Schedule a follow-up.
6. Return to the dashboard to see stale contacts, overdue follow-ups, and suggested actions.

## Current Limitations

- Browser sessions expire without automatic refresh.
- Password reset can request a token, but there is no reset-token screen yet.
- Invitations can be created, but accepting an invite requires API support rather than a complete UI.
