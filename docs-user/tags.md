# Tags

Tags segment contacts.

## How Tags Appear

Tags can be created through the API or by CSV import when the `tags` column contains pipe-separated names, such as:

```text
Investor|Warm Intro|Alumni
```

Contacts display tags in lists and detail pages.

## Current Limitations

- No dedicated tag management UI.
- Contact creation form does not expose tag selection.
- No tag delete/rename UI.
