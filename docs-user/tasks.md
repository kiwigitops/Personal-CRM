# Follow-Ups

Follow-ups are reminders to contact someone later.

## Schedule A Follow-Up

1. Open a contact.
2. Use Schedule follow-up.
3. Pick channel, date, prompt, and optional suggested message.
4. Submit.

The app creates an in-app reminder and queues a follow-up agent job.

## Complete A Follow-Up

Open the Follow-ups page and select Complete.

## Snooze

The API supports snoozing, but the current web follow-up board does not expose a snooze control. Snoozed follow-ups are also hidden from the main board because it only requests pending follow-ups.
