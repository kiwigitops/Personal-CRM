# User Troubleshooting

## I Got Signed Out

Access tokens expire and the web app does not refresh them automatically yet. Sign in again.

## I Requested Password Reset But Cannot Reset

The request page exists. The reset-token entry page does not. Use API tooling until the UI is added.

## My Invite Token Has Nowhere To Go

Invite creation exists, but invite acceptance is not finished in the UI.

## A Company Search Result Goes Nowhere

Company detail pages are not implemented yet.

## My Snoozed Follow-Up Disappeared

The main follow-up page shows pending follow-ups. Snoozed follow-ups are not shown there yet.
