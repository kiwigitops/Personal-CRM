# personal-crm-clients

Frontend monorepo for the Personal CRM product suite.

## Workspaces

- `apps/web`: Next.js SaaS dashboard
- `apps/mobile`: Expo mobile app
- `apps/desktop-linux`: Tauri-based Linux desktop shell
- `packages/ui`: shared design tokens and UI primitives
- `packages/types`: shared application types and schemas
- `packages/api-client`: typed REST API SDK
- `packages/config`: shared TypeScript, Tailwind, lint, and env helpers
- `packages/utils`: shared formatting and business utilities

## Commands

- `npm install`
- `npm run dev:web`
- `npm run dev:mobile`
- `npm run dev:desktop`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
